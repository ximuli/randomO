# Vite 项目部署 Github Pages

## 开启 Github pages 服务

Github仓库 settings -> Pages -> 使用 gh-pages 分支创建 Pages。

## 创建 yml

在项目根目录下创建 .github/workflows 目录，创建一个 .yml 后缀的文件，文件名随意。

将该文件推送至远端仓库。

## 更多
